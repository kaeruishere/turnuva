📱 Proje Hakkında: turnuva

Arkadaş grupları için tasarlanmış, gerçek zamanlı futbol turnuvası platformu.

Temel Özellikler

    Hızlı Katılım: Google Auth ile anında giriş ve profil oluşturma.

    Gelişmiş Lobi: Admin kontrollü başlangıç sistemi ve şifreli/açık oda seçenekleri.

    Akıllı Fikstür: "Circle Method" algoritması ile otomatik, çift maçlı (rövanşlı) lig oluşturma.

    Canlı İstatistikler: Gol krallığı, yenilmezlik serileri ve savunma performanslarını takip eden anlık veri tablosu.

    Otomatik Yönetim: Maç skorları girildikçe güncellenen puan durumu ve otomatik turnuva sonlandırma.

Teknolojiler

    Frontend: Modern JavaScript (ESM), HTML5 ve Glassmorphism (Buzlu cam) efektli CSS3 tasarımı.

    Backend & Veritabanı: Firebase Firestore (Gerçek zamanlı senkronizasyon).

    Oturum Yönetimi: Firebase Auth (Google Login).
